# AetherDB Admin Console

This console provides secure administration features for the AetherDB AI SQL Assistant. It is a password protected management tool responsible for database control, schema insights, AI configuration and safety enforcement.


## Features

### Admin Security
* Protected login screen
* Username and password stored in a .env file
* Logout control with session reset

### System Management
* Connect to supported databases MySQL, PostgreSQL and SQLite
* Refresh schema awareness and generate schema files
* Inspect schema.txt and metadata

### Safety Control
* Toggle destructive SQL operations
* Enable or disable dry run mode

### AI Configuration
* Modify Gemini model version
* Configure maximum schema prompt characters

### Logs and Monitoring
* Real time query statistics
* Status breakdown chart
* Exportable CSV of query history
